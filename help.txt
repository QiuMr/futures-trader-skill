一个用于 Gate.io 期货交易的命令行工具，支持订单管理、持仓查询等功能

使用说明:
  1. 首先使用 save-key 命令保存您的 Gate.io API 密钥
  2. 然后可以使用其他命令进行交易操作
  3. 使用 clear-key 命令清除已保存的密钥

帮助:
  使用 [command] --help 查看具体命令的详细说明

Usage:
  futures-trader [command]

Available Commands:
  account              查询期货账户信息
  cancel-price-orders  批量取消自动订单
  clear-key            清除保存的 API 密钥
  completion           Generate the autocompletion script for the specified shell
  create-order         创建期货订单
  create-trigger-order 创建价格触发订单（止损/止盈单）
  get-price-orders     查询自动订单列表
  help                 Help about any command
  market               查询行情数据
  positions            查询持仓信息
  save-key             保存 Gate.io API 密钥

Flags:
  -h, --help   help for futures-trader

Use "futures-trader [command] --help" for more information about a command.

查询 Gate.io 期货行情数据，包括K线、行情快照、资金费率等

子命令:
  kline      查询K线数据
  ticker     查询行情快照
  funding    查询资金费率

示例:
  # 查询BTC_USDT的K线数据
  futures-trader market kline --contract BTC_USDT --interval 1h --limit 24

  # 查询多个交易对的行情快照
  futures-trader market ticker --contracts BTC_USDT,ETH_USDT

  # 查询资金费率
  futures-trader market funding --contracts BTC_USDT,ETH_USDT

Usage:
  futures-trader market [command]

Available Commands:
  funding     查询资金费率
  kline       查询K线数据
  ticker      查询行情快照

Flags:
  -h, --help   help for market

Use "futures-trader market [command] --help" for more information about a command.

查询指定交易对的K线数据

参数说明:
  - contract: 合约标识，格式为"基础货币_结算货币"，例如: BTC_USDT
  - interval: K线时间间隔，支持: 1m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 12h, 1d, 3d, 7d
  - limit: 获取的K线数量，最大值为2000

示例:
  # 查询1小时K线，24条
  futures-trader market kline --contract BTC_USDT --interval 1h --limit 24

  # 查询1分钟K线，60条
  futures-trader market kline --contract ETH_USDT --interval 1m --limit 60

Usage:
  futures-trader market kline [flags]

Flags:
      --contract string   合约标识 (必填)
  -h, --help              help for kline
      --interval string   K线时间间隔 (默认: 1h) (default "1h")
      --limit int         K线数量 (默认: 100) (default "100")

查询指定交易对的行情快照，包含最新价、涨跌幅、成交量等

参数说明:
  - contract: 合约标识，格式为"基础货币_结算货币"，例如: BTC_USDT
  - contracts: 多个合约标识，用逗号分隔，例如: BTC_USDT,ETH_USDT

示例:

  futures-trader market ticker --contract BTC_USDT

  # 查询多个交易对行情
  futures-trader market ticker --contracts BTC_USDT,ETH_USDT,DOGE_USDT

Usage:
  futures-trader market ticker [flags]

Flags:
      --contract string    合约标识
      --contracts string   多个合约标识，用逗号分隔
  -h, --help               help for ticker

创建期货订单

参数说明:
  - contract: 合约标识，格式为"基础货币_结算货币"，例如: BTC_USDT
  - side: 交易方向，可选值: buy(买入), sell(卖出)
  - size: 订单数量（张）
  - price: 订单价格（可选，市价单可不填）
  - type: 订单类型，可选值: limit(限价单), market(市价单) (默认: limit)
  - reduce_only: 是否仅减仓 (默认: false)

示例:
  # 创建限价买入订单
  futures-trader create-order --contract BTC_USDT --side buy --size 1 --price 70000 --type limit

  # 创建市价卖出订单
  futures-trader create-order --contract BTC_USDT --side sell --size 1 --type market

Usage:
  futures-trader create-order [flags]

Flags:
      --contract string   合约标识 (必填)
      --price float       订单价格 (限价单必填)
      --reduce_only       仅减仓 (默认: false)
      --side string       交易方向: buy 或 sell (必填)
      --size float        订单数量 (张) (必填)
      --type string       订单类型: limit 或 market (默认: "limit")

创建价格触发订单（止损/止盈单）

参数说明:
  - contract: 合约标识，格式为"基础货币_结算货币"，例如: BTC_USDT
  - side: 交易方向，可选值: buy(买入), sell(卖出)
  - size: 订单数量（张）
  - trigger_price: 触发价格 (必填)
  - order_price: 订单价格 (可选，不填则为市价单)
  - type: 订单类型，可选值: limit(限价单), market(市价单) (默认: market)

示例:
  # 创建止损卖出订单
  futures-trader create-trigger-order --contract BTC_USDT --side sell --size 1 --trigger_price 68000

  # 创建止盈买入订单（限价）
  futures-trader create-trigger-order --contract BTC_USDT --side buy --size 1 --trigger_price 75000 --order_price 74500 --type limit

Usage:
  futures-trader create-trigger-order [flags]

Flags:
      --contract string   合约标识 (必填)
      --order_price float 订单价格 (可选)
      --price string      订单价格 (可选)
      --reduce_only       仅减仓 (默认: false)
      --side string       交易方向: buy 或 sell (必填)
      --size float        订单数量 (张) (必填)
      --trigger_price float 触发价格 (必填)
      --type string       订单类型: limit 或 market (默认: "market")
